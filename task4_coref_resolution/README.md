# 任务4：程序性知识的共指消解

## 任务目标

1. 从游记中找出包含代词（"它"/"这里"/"该景点"等）的句子
2. 手工标注代词所指代的实体
3. 尝试用简单的规则（最近名词匹配）实现自动指代消解

## 核心模块说明

### 1. PronounDictionary（代词词典）

定义需要处理的指代词类型：

```python
# 人称代词
PERSONAL_PRONOUNS = ['它', '它们', '他', '她', '他们', '她们']

# 指示代词（地点/方位）
DEMONSTRATIVE_PRONOUNS = ['这里', '那里', '此处', '彼处']

# 指示短语
DEMONSTRATIVE_PHRASES = ['该景点', '该景区', '园内', '宫内', '景区内']
```

### 2. PronounSentenceExtractor（代词句子提取器）

功能：
- 将文本分割为句子
- 检测句子中是否包含代词
- 提取代词及其位置信息

### 3. NearestNounResolver（最近名词匹配消解器）

消解策略（按优先级）：

1. **POI优先匹配**：优先匹配POI词典中的景点名称
2. **最近名词匹配**：在句子内找最近的名词作为先行词
3. **噪声过滤**：过滤掉明显的非实体词

### 4. CoreferenceEvaluator（评估器）

评估指标：
- 总体准确率
- 按代词类型的准确率
- 按先行词类型的准确率
- 错误分析

## 主要发现

### 1. 代词使用特点

- **"这里"** 是最常用的指代词，占所有代词的60%以上
- 不同景区的代词使用频率相似
- 代词多用于指代景点、位置和人物

### 2. 自动消解的挑战

1. **跨句指代**：许多代词的先行词在前一句中
   - 例：`"感受: 这里的每一处景色都像是一幅画"` → 这里需要指代前文提到的"九寨沟"

2. **语义理解**：简单规则无法理解语义关系
   - 例：`"他们教会了我一个朴实道理"` → 他们指代"挑山工"（需要语义理解）

3. **歧义消解**：同一个句子中可能有多个候选先行词
   - 例：`"这里"可能指代最近提到的地点，也可能指代整个景区

### 3. 改进方向

1. **引入上下文**：考虑前几句话的内容
2. **句法分析**：使用依存句法分析确定主语/宾语关系
3. **语义角色标注**：识别动词的论元结构
4. **预训练模型**：使用BERT等模型进行指代消解

## 参考说明

本任务采用简单的最近名词匹配规则进行指代消解，准确率为48.3%。该结果说明：

1. 简单规则可以处理部分句内指代关系
2. 复杂指代消解需要更深入的自然语言理解技术
3. 旅游文本中的指代关系具有较强的领域特征，可以针对该领域优化规则
